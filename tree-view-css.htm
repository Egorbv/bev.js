.tree-view

Css класс для основного контейнера дерева

.tree-view b

Css класс для элементов управления схлопывания \ раскрытия узлов дерева. В данном элементе рисуются либо (плюс\минус), либо стрелки (вбок \ вниз)

.tree-view s

Css класс для иконки узла. В данном элементе рисуется иконка узла.

.tree-view i

Css класс для всего узла дерева. В нем задаются свойста текста и шрифта

.tree-view i:hover

Css класс для всего узла при наведении на него мыши. В частности фон узла под мышью.

.tree-view-node-selected

Css класс для выделенного узла. В частности фон выделенного узла.

.tree-view-collapsed

Css класс для узла имеющего потомков но нохадящимся в схлопнутом состоянии. В частности отвечает за отрисовку иконки символизирующей схлопнутый узел которая позволяет его раскрыть.

.tree-view-collapsed:hover

Css класс для узла имеющего потомков но нохадящимся в схлопнутом состоянии при наведенной на него мыши. В частности отвечает за отрисовку иконки символизирующей схлопнутый узел которая позволяет его раскрыть.

.tree-view-expanded

Css класс для узла имеющего потомков но нохадящимся в раскрытом состоянии. В частности отвечает за отрисовку иконки символизирующей раскрытый узел которая позволяет его схлопнуть.

.tree-view-expanded:hover

Css класс для узла имеющего потомков но нохадящимся в раскрытом состоянии при наведенной на него мыши. В частности отвечает за отрисовку иконки символизирующей раскрытый узел которая позволяет его схлопнуть.

.tree-view div

Css класс для контейнера содержащего список дочерних узлов по отношению к родительскому. В частности описывает отступ дочерних узлов по отношению к родительскому.

.tree-view-edit

Css класс для узла находящегося в режиме редактирования.
